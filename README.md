# UIzapretcombaine
Приложение обёртка для zapret2, стратегий flowseal и tg-ws-proxy от того же flowseal. Предупреждаю код может вести себя аномально т.к является полётом больной фантазии клода и чатгпт. Если вы готовые пойти на риски и понимаете потенциальные проблемы с которыми столкнуться можете его попробовать.
